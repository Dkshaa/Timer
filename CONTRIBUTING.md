# Contributing

Thanks for helping improve Timer.

## Local Setup

1. Clone the repository.
2. Open `index.html` directly, or serve the folder with a static server.
3. Run `npm test` before opening a pull request.

## Pull Request Checklist

- Keep changes focused on one improvement or fix.
- Add or update tests when timer helper behavior changes.
- Update `README.md` or `CHANGELOG.md` when user-facing behavior changes.
